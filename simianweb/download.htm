Windows 2K, 95. nVidia GeForce3

Installation Instructions:
  1. Unzip downloaded file to a directory of your choosing
  2. Make sure you have the correct driver.
    • GeForce3 detonator > v14.20
      v14.20 is recommended
  3. Copy the *glut.dll* file to your windows system32 directory
